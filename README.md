# Insurance Attrition Survival Analysis

## Overview
This project applies survival analysis techniques to estimate 
attrition rates among policyholders in Kenyan financial institutions.

Based on peer-reviewed publication:
**Oburu, J.J. & Simwa, R. (2020).** Estimation of Attrition Rates 
Through Survival Analysis of Financial Institutions in Kenya. 
GSJ Volume 8, Issue 3. ISSN 2320-9186

## Key Methods
- Kaplan-Meier survival estimators
- Cox proportional hazards model
- Log-rank tests for group comparison
- R statistical computing environment

## Key Findings
- Identified critical attrition periods within policyholder lifecycles
- Modelled risk factors driving early policy lapse
- Provided actuarial recommendations for retention strategy

## Tools Used
- R (survival, ggplot2, dplyr)
- Statistical packages: KM curves, Cox PH regression
